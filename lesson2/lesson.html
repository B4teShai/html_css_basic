Lesson 2: CSS Basics

1. CSS Syntax and Selectors

Basic Syntax

selector {
  property: value;
}

Example:

p {
  color: blue;
  font-size: 16px;
}

Common Selectors

  • element - Selects all elements of that type
  • .class - Selects elements with that class
  • #id - Selects element with that ID
  • element.class - Selects elements of that type with that class
  • parent > child - Selects direct children

2. Colors and Typography

Colors

Colors can be specified in several ways:

  • Named colors: red, blue, green
  • Hexadecimal: #ff5733 (This color)
  • RGB: rgb(255, 87, 51)
  • RGBA: rgba(255, 87, 51, 0.5) (with opacity)

Typography

  • font-family: Sets the font
  • font-size: Sets text size
  • font-weight: Sets text boldness
  • line-height: Sets line spacing

3. Box Model

Box Model Components

  • content: The actual content area
  • padding: Space between content and border
  • border: The border around the padding
  • margin: Space outside the border

1. CSS Selectors

Types of Selectors

1. Element Selector: p { color: blue; }

2. Class Selector: .highlight { background: yellow; }

3. ID Selector: #unique { color: red; }

4. Combined Selectors: #special.highlight { ... }

Key Differences:

  • IDs must be unique in the page (only one element can have a specific ID)
  • Classes can be reused on multiple elements
  • ID selectors have higher specificity than classes
  • Use # for IDs and . for classes in CSS
