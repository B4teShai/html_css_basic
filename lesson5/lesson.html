Lesson 5: Advanced Styling

1. CSS Animations

@keyframes and Animation Properties

Bounce
Fade
Rotate
@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } .bounce-animation { animation: bounce 2s infinite; }

Key Animation Properties:

  • animation-name: Name of the @keyframes
  • animation-duration: Length of animation
  • animation-timing-function: Speed curve
  • animation-delay: Start delay
  • animation-iteration-count: Number of times
  • animation-direction: Forward/reverse

2. CSS Transitions

Smooth State Changes

Scale Me
Color Change
.hover-button { transition: all 0.3s ease; } .hover-button:hover { transform: translateY(-5px); box-shadow: 0 5px 15px rgba(0,0,0,0.2); }

Transition Properties:

  • transition-property: Property to animate
  • transition-duration: Length of transition
  • transition-timing-function: Speed curve
  • transition-delay: Start delay

.rotate { transform: rotate(45deg); } .scale { transform: scale(1.5); } .skew { transform: skew(20deg); } .translate { transform: translate(20px, 20px); }

Transform Functions:

  • rotate(), rotateX(), rotateY(), rotateZ()
  • scale(), scaleX(), scaleY()
  • skew(), skewX(), skewY()
  • translate(), translateX(), translateY(), translateZ()
