Lesson 4: Responsive Design

1. Viewport Settings

Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Why it's important:

  • Ensures proper scaling on mobile devices
  • Prevents mobile browsers from using virtual viewports
  • Enables responsive design features

2. Media Queries

Basic Media Query Syntax

@media (max-width: 768px) {
  /* Mobile styles */
}

@media (min-width: 769px) and (max-width: 1024px) {
  /* Tablet styles */
}

Resize your browser to see changes

This box changes color based on screen width:

  • Mobile (≤ 768px): Blue
  • Tablet (769px - 1024px): Green
  • Desktop (≥ 1025px): Purple

3. Responsive Units

Relative Units

REM Units

Based on root font size

EM Units

Based on parent font size

Viewport Units

Based on viewport size

  • rem: Relative to root element
  • em: Relative to parent element
  • vw/vh: Relative to viewport
  • %: Relative to container

4. Responsive Images

Fluid Images

img {
  max-width: 100%;
  height: auto;
}

Picture Element

<picture>
  <source media="(min-width: 768px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="Responsive image">
</picture>

5. Mobile-First Approach

Mobile-First Principles

1. Start Mobile

Design for smallest screens first

2. Progressive Enhancement

Add features for larger screens

3. Performance Focus

Optimize for mobile devices

/* Mobile first base styles */
.element { width: 100%; }

/* Tablet and up */
@media (min-width: 768px) {
  .element { width: 50%; }
}
