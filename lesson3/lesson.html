Common Flex Container Properties:

  • display: flex - Creates a flex container
  • flex-direction - Row or column layout
  • justify-content - Alignment along main axis
  • align-items - Alignment along cross axis
  • gap - Space between flex items

Flex Item Properties

flex-grow: 1
flex-grow: 2
no grow

Common Flex Item Properties:

  • flex-grow - Growth factor
  • flex-shrink - Shrink factor
  • flex-basis - Initial size
  • align-self - Individual alignment

Common Grid Container Properties:

  • display: grid - Creates a grid container
  • grid-template-columns - Defines columns
  • grid-template-rows - Defines rows
  • gap - Space between grid items

4. Positioning

Position Properties

position: relative
position: absolute
position: fixed

Position Values:

  • static - Default positioning
  • relative - Relative to normal position
  • absolute - Relative to positioned parent
  • fixed - Relative to viewport
  • sticky - Hybrid of relative and fixed
